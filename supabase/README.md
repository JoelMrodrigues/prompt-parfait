# Scripts SQL Supabase

Tous les scripts SQL du projet sont regroupés ici. À exécuter dans l’ordre logique (schémas puis migrations) via le **SQL Editor** de Supabase.

- **supabase-schema.sql** — Table `pro_stats` (stats pro / CSV)
- **supabase-team-schema.sql** — Schéma équipe (teams, players, etc.)
- **supabase-team-*.sql** — Tables et migrations liées aux matchs équipe
- **supabase-player-*.sql** — Tables et migrations Solo Q / joueurs
- **supabase-champion-pool-tier.sql** — Pool de champions
- **supabase-team-invite.sql** — Invitations équipe
- **supabase-*-fix*.sql** — Correctifs (RLS, colonnes, etc.)
